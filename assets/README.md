# Brand Assets

Visual identity for NetLogo MCP. The brand concept is **"The Conversational Flock"** — a speech bubble (MCP / AI conversation) containing triangular agents in V-formation (NetLogo's turtles, emergence). All assets were generated from the spec in [`ai-image-brief.json`](ai-image-brief.json), which is the source of truth for regenerating or extending the set.

| Role | Hex | Name |
